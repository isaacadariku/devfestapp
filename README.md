# devfestapp

A Web App for Abuja DevFest 2022, powered by Flutter. Showcasing the speaker's profile, social links and slide url.

## Getting Started

Clone the repository

```bash
git clone
```

Install dependencies

```bash
flutter pub get
```

Run the app

```bash
flutter run
```
